Welcome to NoSQL CRUD API

This is a simple API built with Express.js to perform CRUD operations on a NoSQL database MongoDB with GIT Version Controlled CI-CD.

Available Routes:

  • GET / - Home route
  • GET /api/v1/home - Current page route
  • GET /api/v1/health - Health check
  • GET /api/v1/users - Retrieve all users
  • POST /api/v1/users - Create a new user
